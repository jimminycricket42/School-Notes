# Relational Operators
| Symbol | Meaning                   |
| ------ | ------------------------- |
| >      | Greater Than              |
| <      | Less Than                 |
| =      | Equal To                  |
| !=     | Not Equal TO              |
| >=     | Greater Than and Equal to |
| <=     | Less than and Equal To    |
